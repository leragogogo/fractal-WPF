Если не очень понял/a почему в таком диапазоне углы для дерева, пожалуйста, посмотри картинки, которые лежат рядом с решением.

При изменении глубины рекурсии, фрактал автоматически перерисовывается.
Чтобы фрактал перерисовался при изменении коэффицента, углов или расстояния между отрезками нужно переключить фокус.

Нулевой глубине рекурсии соответствует просто изначальная линия, квадрат или треугольник. 

Для множества Кантера: расстояние между отрезками я считаю от верхних границ отрезков, толщина отрезкров фиксированная - 0.5, ширина первого отрезка фиксированная - 9, в процессе рендеринга сцена автоматически маштабируется под ширину и высоту канваса.

Реализовано 2 допа: автоматическая перерисовка фрактала при изменении размеров окна, сохранение фрактала в виде картинки.
